Что бы начать использовать pyflex, ваша программа должна создать класс, унаследованный от
To start using pyflex in your program you should create inherited from pyflex.app.PyFlexApp class. 
        from pyflex.app import PyFlexApp

        class MyApp(PyFlexApp):
            def __init__(self):
                super().__init__()
Please draw your attention on the __init__ method. If you want to inherit it you should write super().__init__() at the first line of your initialization method.
